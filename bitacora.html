Proceso de Desarrollo de la Propuesta de Analítica de Datos

Descripción del proceso realizado para el desarrollo de la actividad académica

Inicio de la actividad

El desarrollo de esta actividad comenzó con la revisión de las instrucciones publicadas en la plataforma académica. En el caso de estudio se presenta una empresa que se encuentra en un proceso de migración de información acumulada durante aproximadamente veinte años.

Gran parte de esta información se encuentra almacenada en diferentes formatos y no está organizada de manera estructurada. Esta situación dificulta el análisis de los datos y limita su utilización para apoyar la toma de decisiones dentro de la empresa.

Por esta razón, el objetivo de la actividad consiste en proponer una forma de organizar y analizar los datos mediante el uso de herramientas de analítica que permitan aprovechar mejor la información disponible.

Investigación sobre analítica de datos

En una segunda etapa se realizó una investigación sobre el uso de herramientas de analítica de datos utilizadas por las empresas para analizar grandes volúmenes de información.

Durante esta etapa se revisaron diferentes fuentes académicas y recursos educativos con el objetivo de comprender cómo las organizaciones utilizan la información para mejorar sus procesos y fortalecer sus estrategias comerciales.

Esta investigación permitió identificar varias herramientas tecnológicas que facilitan el análisis de datos y la generación de reportes.

Selección de herramientas de análisis

Después de analizar diferentes opciones se seleccionaron algunas herramientas de analítica de datos que pueden ayudar a la empresa a organizar su información y generar análisis más claros.

  • Power BI para la creación de paneles y reportes interactivos.
  • Tableau para la visualización de datos mediante gráficos.
  • Microsoft Excel para organizar información y realizar cálculos.
  • Google Analytics para analizar el comportamiento de los usuarios.
  • Python como herramienta para el procesamiento y análisis de datos.

Estas herramientas permiten procesar información de manera eficiente y presentar los resultados de forma clara para facilitar la toma de decisiones.

Desarrollo de la propuesta

Con base en la información recopilada se elaboró una propuesta para mejorar el manejo de los datos dentro de la empresa. Esta propuesta incluye un proceso que contempla la recolección de información, la limpieza de datos, la organización de bases de datos y el análisis mediante herramientas tecnológicas.

El objetivo principal es transformar los datos disponibles en información útil que permita identificar oportunidades de crecimiento y mejorar el desempeño del área comercial.

Creación de la página web

Como parte del desarrollo de la actividad se creó una página web con el fin de presentar la propuesta elaborada por el grupo. En esta página se incluye la descripción del problema, el análisis realizado, las herramientas propuestas y las conclusiones del trabajo.

La página fue desarrollada utilizando HTML para la estructura del contenido y CSS para mejorar la presentación visual de la información. Posteriormente el proyecto fue publicado en un repositorio para facilitar su visualización a través de internet.

Conclusiones del proceso

El desarrollo de esta actividad permitió comprender la importancia de la analítica de datos en las organizaciones actuales. El análisis adecuado de la información permite mejorar la toma de decisiones, optimizar procesos y fortalecer el crecimiento empresarial.

Además, la creación de la página web permitió presentar el contenido de la actividad de una manera organizada y accesible para los usuarios.
